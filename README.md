# Weather

郵便番号から住所を表示し、その地域の天気予報を確認できるシンプルなWebアプリです。

公開URL: https://mukumuku-bot.github.io/weather/

## 機能

- 高松と東京の天気予報を初期表示
- 郵便番号を入力して住所を検索
- 検索した住所の地域の天気予報を表示
- 今日の気温、降水確率、明日の気温、風を表示
- 派手な背景と大きなアニメーション付きのデザイン

## 使い方

1. ページを開きます。
2. 郵便番号を7桁で入力します。
3. 「検索」を押すと、住所とその地域の天気予報が表示されます。
4. 「更新」を押すと、表示中の地域の天気を再取得します。

## 使用データ

- 住所検索: ZipCloud
- 天気予報: 気象庁

## ローカルで見る方法

このフォルダで簡易サーバーを起動して、ブラウザで開きます。

```bash
python -m http.server 8000
```

その後、以下にアクセスします。

```text
http://localhost:8000/
```
