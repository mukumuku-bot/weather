# Weather

郵便番号から住所を表示し、その地点の天気予報を確認できるシンプルな Web アプリです。

公開 URL: https://mukumuku-bot.github.io/weather/

## できること

- 高松と東京の天気を初期表示
- 郵便番号から住所を検索
- 検索した住所の天気を表示
- ランダムに住所を選んで、その地点の天気を表示
- 今日の気温、降水確率、明日の気温、風を表示

## 使い方

1. ページを開きます。
2. 郵便番号を 7 桁で入力するか、`ランダム住所` を押します。
3. `検索` を押すと、その住所の天気を表示します。
4. `更新` を押すと、表示中の天気を再取得します。

## 利用データ

- 住所検索: ZipCloud
- 天気予報: 気象庁

## ローカル確認

```bash
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000/` を開いて確認してください。
